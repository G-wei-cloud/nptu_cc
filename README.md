#### git github
### Use git clone this program:
```sh
# git clone https://github.com/G-wei-cloud/nptu_cc 
```

### check if the file different:
```sh
# git status
```

### add index on file:
```sh
# git add <file name>
```

### upload file:
```sh
# git push
```

#### docker
### Start service:
```sh
$ docker-compose up -d
```

### restart service:
```sh
$ docker-compose restart
```

### check all container:
```sh
$ docker ps or $ docker ps -a
```

### check all :
```sh
$ docker-compose up -d
```

### Stop service:
```sh
$ docker-compose down
```

### Access a running conto=ainer
```sh
$ docker-compose exer $SERVICE_NAME bash
```
